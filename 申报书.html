一、项目背景与意义

TOML (Tom’s Obvious, Minimal Language) 是一种广泛使用的配置文件格式，被 Rust、Python、Go 等语言的生态广泛采用。目前 MoonBit 语言生态中尚无成熟的 TOML 解析库。

本项目旨在为 MoonBit 提供一个完整、高效、易用的 TOML 解析与序列化库，填补 MoonBit 在配置文件处理领域的基础设施空白，助力 MoonBit 生态的完善。

二、技术方案

2.1 核心架构

┌─────────────────────────────────────────────┐
│                  moonbit-toml                │
├─────────────────────────────────────────────┤
│  parse_toml(input: String)                  │
│       → Map[String, TomlValue]              │
│                                             │
│  ┌───────────┐  ┌──────────┐  ┌───────────┐ │
│  │ ParseState│→ │ Lexer    │→ │  Parser   │ │
│  │ (位置管理) │  │ (词法分析)│  │ (语法分析)│ │
│  └───────────┘  └──────────┘  └───────────┘ │
├─────────────────────────────────────────────┤
│              TomlValue 数据类型              │
│  Str | Int | Float | Bool | Array | Table   │
└─────────────────────────────────────────────┘

2.2 已实现功能

  • ✅ 基础键值对解析（字符串、整数、浮点数、布尔值）
  • ✅ 表解析（[section]）
  • ✅ 嵌套表（[server.production]）
  • ✅ 数组解析（[1, 2, "mixed"]）
  • ✅ 注释处理（# comments）
  • ✅ 字符串转义（\n, \t, \", \\）
  • ✅ 7 个白盒单元测试

2.3 技术亮点

  1. 零拷贝设计：利用 MoonBit 的 StringView 实现高效字符串处理
  2. 函数式风格：不可变状态传递，易于推理和测试
  3. 类型安全：完整的 TomlValue 代数数据类型，编译期类型检查
